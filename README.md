#划词翻译
![](http://eleveni386.7axu.com/image/youdao.png)

#特性
1. 鼠标选词. 在鼠标所在位置出现翻译框
2. 托盘图标. 右键托盘图标, 设置开启划词或者关闭划词
3. 全局热键. `Ctrl+F8` 开启或者关闭划词
4. 支持pdf, 网页

#依赖
python-Xlib
requests
python-webkit
python-gtk

#安装
修改openyoudao.desktop中的相关路径

修改项包括但不仅限于:
`Exec=`
`Icon=`

以你本地实际路径为准. 将修改好的openyoudao.desktop 放置 `/usr/share/applications/` 下即可
